# Zwicky Box (Morphological Analysis)

Generate creative solutions through systematic combination.

## When to Use

| Trigger                    | Use This Tool             |
| -------------------------- | ------------------------- |
| Stuck on obvious solutions | Force novel combinations  |
| Need many options quickly  | Systematic generation     |
| Innovation or pivot needed | Explore possibility space |

## The Framework

```
┌────────────────┬─────────────┬─────────────┬─────────────┐
│ Dimension      │ Option A    │ Option B    │ Option C    │
├────────────────┼─────────────┼─────────────┼─────────────┤
│ [Parameter 1]  │ [value]     │ [value]     │ [value]     │
│ [Parameter 2]  │ [value]     │ [value]     │ [value]     │
│ [Parameter 3]  │ [value]     │ [value]     │ [value]     │
│ [Parameter 4]  │ [value]     │ [value]     │ [value]     │
└────────────────┴─────────────┴─────────────┴─────────────┘

Total combinations = A × B × C × D options
Pick one from each row → novel solution
```

## How to Apply

1. **Define the problem** clearly
2. **List parameters** (4-6 dimensions that define solutions)
3. **List options** for each parameter (3-5 options per dimension)
4. **Generate combinations** by picking one from each row
5. **Evaluate** most promising combinations
6. **Force unusual combos** to spark innovation

## Example

**Problem:** New revenue stream for developer tools company

```
┌─────────────────┬──────────────┬──────────────┬──────────────┐
│ Dimension       │ Option A     │ Option B     │ Option C     │
├─────────────────┼──────────────┼──────────────┼──────────────┤
│ Customer        │ Individual   │ Teams        │ Enterprise   │
│ Revenue Model   │ Subscription │ Usage-based  │ Marketplace  │
│ Delivery        │ Cloud SaaS   │ Self-hosted  │ IDE Plugin   │
│ Value Prop      │ Speed        │ Quality      │ Compliance   │
└─────────────────┴──────────────┴──────────────┴──────────────┘

Total combinations: 3 × 3 × 3 × 3 = 81 options

Sample combinations:
┌──────┬────────────┬─────────────┬────────────┬────────────┐
│ #    │ Customer   │ Revenue     │ Delivery   │ Value      │
├──────┼────────────┼─────────────┼────────────┼────────────┤
│ 1    │ Individual │ Subscription│ Cloud SaaS │ Speed      │
│      │ (Status quo - current product)        │            │
├──────┼────────────┼─────────────┼────────────┼────────────┤
│ 2    │ Enterprise │ Usage-based │ Self-hosted│ Compliance │
│      │ (New: on-prem compliance tool, pay per scan)       │
├──────┼────────────┼─────────────┼────────────┼────────────┤
│ 3    │ Teams      │ Marketplace │ IDE Plugin │ Quality    │
│      │ (New: team-curated plugin marketplace)│            │
└──────┴────────────┴─────────────┴────────────┴────────────┘
```

**Insight:** Combination #2 (enterprise compliance) was non-obvious but addresses real market gap.

## Key Insight

The box forces you past the first obvious answer. Most innovation comes from unusual combinations that you wouldn't naturally consider.
