# Thinking Tools Catalog

Quick reference for selecting the right mental model.

---

## Strategic Frameworks

Higher-level frameworks for complex situations. Use before diving into tactical tools.

| Framework                              | Purpose                             | When to Use                                   |
| -------------------------------------- | ----------------------------------- | --------------------------------------------- |
| [Handshake](handshake.md)              | Drive action from communication     | Meetings, negotiations, getting buy-in        |
| [SCOPE](scope.md)                      | Right-size analysis before starting | Research, investigation, any analysis work    |
| [Leverage Points](leverage-points.md) | Find where to intervene in systems  | Complex system change, choosing interventions |
| [SPOT](spot.md)                        | Surface and act on recurring patterns | Retrospectives, debugging recurring issues  |

---

## Default Tools

For most situations, use these first:

| Situation              | Default                                      | When to Use                            |
| ---------------------- | -------------------------------------------- | -------------------------------------- |
| Prioritizing work      | [Impact-Effort](tools/impact-effort.md)      | Backlog grooming, what to do next      |
| Breaking down problems | [Issue Trees](tools/issue-trees.md)          | Complex problems, exhaustive analysis  |
| Finding root cause     | [Ishikawa](tools/ishikawa.md)                | Debugging, incidents, postmortems      |
| Making decisions       | [Decision Matrix](tools/decision-matrix.md)  | Multi-option choices with tradeoffs    |
| Understanding systems  | [Feedback Loops](tools/feedback-loops.md)    | Architecture, metrics, consequences    |
| Communicating clearly  | [Minto Pyramid](tools/minto-pyramid.md)      | Writing, presentations, exec summaries |
