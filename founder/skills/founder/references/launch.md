# Plan Launch

Create a 90-day launch execution plan with clear weekly actions across three phases.

## Input Required

Ask for (if not provided):

- Product name and one-liner
- Target customer (who are you launching to?)
- Launch channels you're considering (Product Hunt, Reddit, Twitter, etc.)
- Current state (MVP ready? Landing page? Email list?)
- Resources available (budget, team size, time per week)

## Rules

- Tasks must be actionable by a solo founder
- Budget constraints should inform channel selection
- Use ask tool to understand resources before planning
