# moo.md

Thoughtful plugins for Claude Code.

## What's Here

Three plugins.

| Plugin        | Purpose                                                                               | Install  |
| ------------- | ------------------------------------------------------------------------------------- | -------- |
| **hope**      | Cognitive operating system. Clarify intent, state confidence, identify failure modes. | Required |
| **product**   | PRD generation, competitive analysis, research synthesis, goal systems.               | Optional |
| **wordsmith** | Voice extraction, precision editing, narrative structure, copy evaluation.            | Optional |
